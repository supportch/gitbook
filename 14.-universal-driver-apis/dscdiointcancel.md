# dscDIOIntCancel

```c
BYTE dscDIOIntCancel(DSCB board);
```

This function is used to cancel the digital interrupts.

{% tabs %}
{% tab title="Input Parameters" %}
| Name | Description |
| :--- | :--- |
| DSCB | The handle of the board to operate on |
{% endtab %}
{% endtabs %}

| Return Value |
| :--- |
| Error code or 0. |
