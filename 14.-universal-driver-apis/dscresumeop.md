# dscResumeOp

```c
BYTE dscResumeOp(DSCB board)
```

This function resumes A/D interrupts from the point at which they were paused.

{% tabs %}
{% tab title="Input Parameters" %}
| **Name** | **Description** |
| :--- | :--- |
| DSCB  | The handle of the board to operate on |
{% endtab %}
{% endtabs %}

| Return Value |
| :--- |
| Error code or 0. |
